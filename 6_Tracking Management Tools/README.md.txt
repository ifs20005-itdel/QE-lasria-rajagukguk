Tracking Management Tools
\> sebuah software yang digunakan dalam suatu proyek atau development software yang agar berjalan eisien dan efektif.

Contoh:
-JIRA
\>Dikembangkan oleh perusahaan perangkat lunak Atlanssian sebagai alat bantu untuk
-Bug Tracking
-Issue Tracking
-Project Management
  
7 Point For Creating Issue
-Project -> Dipilih sesuai dengan issues yang akan dibuat dan di tempatkan pada 	project tersebut.
-Issues Type 
1. Story ->Jenis issue yang biasa digunakan untuk membuat fitur baru.
2. Task ->Jenis issue yang biasa digunakan untuk melakukan perincian tugas-tugas 	yang dikerjakan.
3.Bug -> Jenis issue ang digunakan jika terjadi oenemuan bug saat proses testing 	oleh QE
4. Epic -> Jenis issue untuk pengelompokan task.
- Reporter -> tentukan reporter berdasarkan yang membuat tugas tersebut
-Description -> menuliskan dekripsi dengan jelas menggunakan format yang sama dengan dituliskan pada pivotal tracker.
-Priority -> ditentukan berdasarkan dari tingkat kesulitan sebuah task atau issue.
-Assignee -> ditentukan berdasarkan dari orang yang akan bertanggung jawab mengerjakan tugas .
-Sprint -> ditentukan berdasarkn dari ruang waktu pengerjaan sebuah task atau issue.
